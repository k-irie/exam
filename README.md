### 表紙ページ

+ 解答ページへ移動する
+ 答え合わせページへ移動する\
ただし、回答ページで未回答の問題があると答え合わせページへ移動できないこととする


### 解答ページ

+ 表紙ページへ移動する
+ 答え合わせページへは移動出来ないようにする
+ 基本的に最初の問題を出題する
+ 未解答の問題があった場合、そこから始める
+ 全ての問題を解答しないと「答え合わせ」は無効とする
+ 4択の中から1つを選択する
+ 選択しないと次のページへは進めない
+ 次の問題へ移動する \
→ この時解答をanswerテーブルに保存する
 

### 答え合わせページ

+ 表紙ページへ移動する
+ 解答ページへ移動する
+ 解答した結果を教示する
+ 正解と解説を表示する
+ 確認のため「前へ」と「次へ」ボタンが有り見直す事が可能
+ 最初の問題は「前へ」、最後の問題では「次へ」を無効化する
